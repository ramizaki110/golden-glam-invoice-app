Golden Glam Hosted Web App

Files to deploy together:
- GoldenGlam_InvoiceGenerator_hosted.html
- golden_glam_invoice_webapp.py
- golden_glam_invoice_generator.py
- golden_glam_logo_final.png
- requirements_hosted.txt
- render.yaml

Fastest deployment on Render:
1. Create a new GitHub repo.
2. Upload the 6 files above.
3. In Render, create a new Web Service from that repo.
4. Render will detect render.yaml automatically.
5. Once deployed, share the app URL with colleagues.

What colleagues do:
- Open the URL
- Fill the invoice
- Click Generate PDF + Excel
- Both files download automatically
