📋 How to generate an invoice
Step 1 — Fill in the invoice
  1. Go to the New Invoice tab — fill in invoice number, date, and optional reference
  2. Select or fill in Client details
  3. Click + Add item for each product — description, item no., vendor, qty, price, discount, and optional photo
  4. Set delivery charge, sales tax, delivery type, and payment terms
  5. Add any notes to appear on the invoice
⚡ Option A — Direct from this app
  1. Come to the Summary & Export tab
  2. Click Refresh to build the summary
  3. Click Generate PDF + Excel
  4. Both files download automatically
✅ Fastest — no Claude needed
🤖 Option B — Via Claude
  1. Click Refresh then Copy
  2. Open a new Claude chat
  3. Upload golden_glam_invoice_generator.py + golden_glam_logo_final.png
  4. Paste the summary and send:
    Generate the PDF invoice from this summary using my script
  5. Download both files Claude returns
